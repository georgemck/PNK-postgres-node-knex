# PNK-postgres-node-knex
Example of PostgreSQL Database using Knex with NodeJS

Tested with NodeJS versions 22.14.0 and 18.20.7


package.json setting for type must match with code

CommonJS:
    package_commonjs.json
    "type":"commonjs" 
    node common.js 

Module:
    package_module.json
    "type":"module" 
    node module.mjs
